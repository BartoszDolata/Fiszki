# Fiszki
 Projekt PAI kl4 

Dokumentacja:
-projekt zakłada stworzenie strony internetowej do nauki jezyka angielskiego technicznego
-Wystepuje baza danych przechowujaca fiszki, uzytkownikow, ich statusy itp.

Funkcjonalność:
-Administrator może:
    -tworzyć użytkowników;
    -usuwać użytkowników;
    -tworzyć zestawy;
    -usuwać zestawy;
    -dodawać fiszki;
    -usuwać uzytkownikow;
-Uczen:
    -przegląda zestawy;
    -uczy się;
    -dodaje fiszki;
    -dodaje zestawy;
    -usuwa swoje zestawy;
-Nauczyciel:
    -dodaje uczniow;
    -usuwa uczniow;
    -dodaje zestawy;
    -usuwa uczniów
    -dodaje fiszki;
    -usuwa fiszki;
    -dodaje klasy;
    -usuwa klasy;
